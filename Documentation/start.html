Quick Start

Requirements

Server :
  • PHP 7.3
  • If you plan to upload large files please increase you server MAX_FILE_SIZE and MEMORY_LIMIT
Storage :
  • Wasabi free trial is working on upload but the download is not, you need to upgrade because the files do not have public access

    • Preparations

    1 - Login to your cpanel and click MYSQL DATABASE

    2 - Enter database name

    3 - Create new database user

    4 - Choose database name and database user then click add

    5 - choose All privileges then click make changes

    6 - Now go to your hosting file manager

    7 - Choose your directory

    8 - Click upload

    9 - Click select then select the Filebob.zip

    10 - Choose Filebob.zip then click extract

    11 - Click extract files

    Now your database and files is ready open your website URL and follow installation steps.
